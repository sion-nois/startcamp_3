



[페르소나]
너는 바닐라 HTML/CSS/JS 프론트엔드 개발자야. 프레임워크·빌드도구 없이 작업해줘.

[기획서]

| 구분 | 기획 및 구현 내용 |
| :--- | :--- |
| **1) 메인 컬러와 폰트** | • **배경색:** 짙은 네이비 (`#172554` / `oklch(28.2% 0.091 267.935)`) <br>• **글꼴:** 깔끔하고 가독성 높은 **Pretendard** |
| **2) 타겟 방문자** | • 현재 함께 일하고 있는 **나의 팀 동료들** |
| **3) 어조와 분위기** | • 정중하면서도 벽이 없는 **친근한 존댓말 톤** |
| **4) 전체적인 UI 구성** | • **우주 지도(Galaxy Map) 인터랙티브 스페이스**<br>• 첫 화면에 5개의 행성이 한눈에 조망됨<br>• 특정 행성 클릭 시, 해당 행성으로 **클로즈업(Zoom-in)되며 상세 정보가 노출**되는 방식 |
| **5) 카테고리 (5개 행성)**| **1. 프로필 행성:** 이름, 나이 등 기본 인적 사항<br>**2. 취미 행성:** 개인적인 관심사와 즐기는 활동<br>**3. 기술 행성:** 업무적 역량 및 보유 스킬<br>**4. 동물 행성:** 좋아하는 동물 또는 반려견/반려묘 소개<br>**5. 기타 행성:** 그 외 동료들에게 공유하고 싶은 자유로운 이야기 |

[요청]
기획서 기반으로 1페이지 웹사이트를 만들어줘.
구조: 첫 화면에는 히어로섹션이 나와 문구는 "Welcome to my space!"야. 이후 구조는 기획서의 "전체적인 UI 구성"을 따라가줘
톤과 문구는 기획서의 "어조와 분위기"를 따라가줘.

[형식]
Canvas에서 바로 미리보기 가능하게. 전체 구조 먼저 만들고, 이후 디테일 다듬기.
